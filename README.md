# Car Market Analytics Dashboard

Developed a comprehensive web-based analytics platform for visualizing and analyzing used car market data using React and modern web technologies. The application processes and presents market insights from a dataset of vehicles, enabling data-driven decision-making for car market trends.

Key Features:

Interactive dashboard with multi-view navigation (Dashboard and Highlighted sections)
Advanced search and filtering capabilities for brands and models
Real-time data aggregation calculating total inventory and market values by brand/model
Data visualization using Chart.js with pie charts and stacked bar charts for market distribution analysis
Responsive UI built with React Bootstrap for optimal user experience across devices
Client-side data processing handling thousands of vehicle records efficiently
Technical Stack:

Frontend: React 18, React Router for SPA navigation
UI Framework: React Bootstrap for responsive design
Data Visualization: Chart.js and React-ChartJS-2
Build Tool: Vite for optimized development and production builds
Styling: Custom CSS with Bootstrap integration
Package Management: pnpm
Technical Highlights:

Implemented complex data aggregation algorithms for calculating brand/model statistics
Optimized performance for handling large JSON datasets
Created reusable component architecture for maintainability
Deployed as a live web application accessible via GitHub Pages
